# ToDo Console App

A toDo console app consumes todo Web service apis

## Table of contents

- [Overview](#overview)
    - [Built with](#built-with)

## Overview

1) Clone the project
   ``` https://github.com/MohamedEldefrawy/ToDo-Client.git```
2) in the project directory run the following
    ```
   mvn clean install
   ```
